# U–Th Radiometric Age Dating

Pipeline for computing Uranium-Thorium ages from carbonate samples (speleothems, corals, travertines) using mass spectrometry output. Ages up to ~500 ka with full Monte Carlo uncertainty propagation.

## Requirements

```
numpy
pandas
matplotlib
seaborn
hickle
tqdm
```

```bash
pip install numpy pandas matplotlib seaborn hickle tqdm
```

## Quick Start

1. Open `UTh_age_v2.ipynb` in Jupyter.
2. Edit **only the Configuration cell** near the top:
   - Set `CSV_PATH` to your input file (or switch to `INPUT_MODE = "manual"` for a single sample).
   - Adjust `N_WALKS` (default 10 000; use 100 for a quick test).
   - Set `OUTPUT_STEM` to control output file names.
3. Run all cells top-to-bottom (`Kernel → Restart & Run All`).

Outputs are written to the same directory:

| File | Contents |
|------|----------|
| `results.csv` | Ages, ±2σ, concentrations, δ²³⁴U measured & initial, reliability flag |
| `results.xlsx` | Same + second sheet with full input data |
| `results.pdf` | Title page + results table + one 3-panel diagnostic figure per sample |

## Input CSV Format

Two formats are accepted automatically by `pipeline.load_csv()`.

### Old format — single-row flat header

```
Sample, 234/238_U, 234/238_Uer, 232/230_Th, 232/230_Ther,
[238U](pmol_g), [238U]_er, [230Th](pmol_g), [230Th]_er
```

### New format — two-row hierarchical header

Columns are isotope ratio names at level 0, with `Average` / `uc (k=1)` sub-columns at level 1.
See `Prueba_repro_ U-Th dating.csv` for a working example of the new format.

After loading, the DataFrame contains normalised columns:

| Column | Description |
|--------|-------------|
| `U238_ppb` | ²³⁸U concentration (ppb) |
| `U238_rsd` | Relative standard deviation |
| `Th230_pmolg` | ²³⁰Th concentration (pmol/g) |
| `Th230_pmolg_err` | Absolute 1σ error |
| `Th232_pmol` | ²³²Th concentration (pmol/g) derived from ²³²Th/²³⁰Th ratio |
| `Th232_err` | Absolute 1σ error |
| `delta_234U` | δ²³⁴U in permil (‰) |
| `delta_234U_err` | Absolute 1σ error |

## Diagnostic Output

Each sample produces a 3-panel figure:

- **Panel A** — Monte Carlo age distribution (KDE) with mean ± 2σ
- **Panel B** — ²³⁰Th/²³⁸U ingrowth curves for δ²³⁴U = 0 and the measured value
- **Panel C** — Isochron diagram with age isolines, back-evolution path, and sample point

The isochron diagram requires three pre-computed grid files (`*.hkl`, ~960 MB each) that must be in the same directory as the notebook. These are generated by `U_Th_gridding.ipynb` (slow; only needed if decay constants change).

## Reliability Flag

Samples with a ²³⁰Th/²³²Th activity ratio below 20 are flagged with ⚠ in the output. When this ratio is low, the detrital ²³⁰Th correction dominates and the reported age uncertainty may be underestimated.

## Module Overview

```
isotope_constants.py          Decay constants (λ₂₃₀, λ₂₃₄, λ₂₃₈, λ₂₃₂), atomic masses,
                              and solver configuration (MAX_AGE, convergence threshold,
                              detrital correction parameters)

calc_Th230_U238_act.py        Core physics — computes ²³⁰Th/²³⁸U activity ratio from
                              δ²³⁴U and age (closed-system ingrowth equation)

age_calc_Th230_U238_CS.py     Iterative age solver — steps ±1 yr until computed activity
                              ratio matches measured value; returns None if age > MAX_AGE

m_Th230_U238_a_from_concs.py Converts pmol/g + ppb concentrations to activity ratio

workers.py                    Measurement namedtuple and Monte Carlo worker function —
                              randomises inputs, applies detrital correction, calls solver

pipeline.py                   load_csv() — auto-detects and normalises both CSV formats;
                              run_mc() — runs workers via multiprocessing for one sample
```

## Methods

Ages are calculated from the ²³⁰Th/²³⁸U disequilibrium method assuming a closed carbonate system. Uncertainties are propagated by Monte Carlo simulation (default 10 000 walks). Detrital ²³⁰Th correction is applied with a 50% (2σ) uncertainty on the initial ²³⁰Th/²³²Th activity ratio (secular equilibrium value: 4.4 × 10⁻⁶).

Decay constants follow Cheng et al. (2013) *Quaternary Geochronology* 20, 142–158.
